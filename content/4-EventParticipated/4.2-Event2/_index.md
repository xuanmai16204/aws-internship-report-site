# SUMMARY REPORT
# Meet Up – First Cloud AI Journey

### 1. General Information
- **Event Name:** Meet Up – First Cloud AI Journey
- **Date & Time:** May 30, 2026
- **Location:** Amazon Web Services Viet Nam Company Limited Office
- **Role:** Participant (Attendee)

### 2. Event Objectives
The Meet Up aimed to provide a platform for intern groups to exchange progress reports, share hands-on implementation experiences, and resolve system integration difficulties. The event focused heavily on frontend/backend connectivity, cloud deployment configurations, and resource optimization on AWS.

### 3. Key Activities
During the Meet Up, I actively participated in project updates and technical discussions regarding the CloudDoc project structure:
- **Presenting CloudDoc Progress**: Demonstrated the completed React interfaces for the Home, Search, Upload, and Administration Dashboard modules.
- **Discussing Frontend/Backend Integration**: Discussed details of how document metadata is serialized in the DB and how secure API payloads should connect the client with the Node.js backend server.
- **AWS Deployment Consultations**: Consulted with AWS mentors on deployment strategies for hosting the backend on EC2, setting up Application Load Balancers, and formatting security via IAM policies.
- **Interactive Q&A Session**: Asked and received actionable answers from AWS engineers regarding securing raw endpoints and managing CORS configurations for uploads to Amazon S3.

### 4. Knowledge Gained
From this Meet Up session, I acquired practical development and architecture insights:
- **System Synchronization Experience**: Gained a clear understanding of how client interfaces and server APIs synchronize document metadata before and after saving logs.
- **IAM Policies and Least Privilege**: Learned how to define scoped security rules using IAM policies to shield AWS resources from unauthorized actions.
- **CORS Configuration for S3**: Understood how to configure Cross-Origin Resource Sharing (CORS) rules on the S3 bucket to allow secure React client file uploads via presigned URLs.

### 5. Personal Reflection
The Meet Up served as a great review mechanism to validate our CloudDoc designs against industry standards. The guidance from our mentors helped resolve key blockers in system architecture and CORS settings. These practical lessons were instrumental in keeping our project integration on track while teaching me proper systematic debugging approaches.
